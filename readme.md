# DevOps Course Project

University project for CI/CD course at UCU (Ukrainian Catholic University).

## Projects

### 1. Infrastructure as Code (Terraform)
IaC setup using Terraform for provisioning cloud infrastructure.

### 2. Local LLM Chat (`local_llm_chat/`)
Chat application with local LLM integration.

**Topics covered:**
- Docker containerization
- Docker compose for networking multiple services
- Kubernetes deployment (local)
- Service orchestration

### 3. Todo App (`react-express-mongodb/`)
Full-stack application: React frontend + Express backend + MongoDB.

**Topics covered:**
- Docker multi-stage builds
- Kubernetes on EKS (AWS)
- CI with GitHub Actions (tests, build, push to ECR)
- CD with ArgoCD (GitOps)
- Blue-green deployments with Kustomize

## Technologies

- **Containerization**: Docker
- **Orchestration**: Kubernetes (Minikube, EKS)
- **IaC**: Terraform
- **CI**: GitHub Actions
- **CD**: ArgoCD
- **Cloud**: AWS (ECR, EKS)
- **Configuration Management**: Kustomize

## Course Focus

CI/CD pipelines, GitOps practices, and cloud-native deployments.
